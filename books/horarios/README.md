# Horários — Histórias que se Cruzam

Fichas individuais por **período do dia**. Cada ficha define faixa horária canônica, função emocional, atmosfera, personagens predominantes e regras de uso.

**Master tracker:** [`../biblia-continuidade-horarios.md`](../biblia-continuidade-horarios.md).

## Princípio essencial

> Toda cena tem horário registrado. Cenas compartilhadas têm horário travado. Vocabulário fixo.

## Lista de períodos

| Período | Faixa | Função predominante |
|---|---|---|
| [Madrugada](madrugada.md) | 00h–05h | Decisões íntimas, intensidade |
| [Amanhecer](amanhecer.md) | 05h–07h | Recomeço pequeno, convergência final |
| [Manhã](manha.md) | 07h–12h | Cotidiano, gestos novos |
| [Meio-dia](meio-dia.md) | 12h–14h | Verdade nua, sombra dura |
| [Tarde](tarde.md) | 14h–17h | Movimento estável, trabalho |
| [Entardecer](entardecer.md) | 17h–19h | Cair da tarde, golden hour, encontros |
| [Noite](noite.md) | 19h–00h | Vida noturna, frio seco, presença |
| [Horários canônicos](horarios-canonicos.md) | hora exata | 3h16, sino, anúncios |

## Estrutura de cada ficha

1. Identidade do período
2. Faixa horária canônica
3. Função emocional
4. Atmosfera (luz, sons, sensações)
5. Personagens predominantes
6. Cenas-chave neste período
7. Conexão com Antigo, Personagem Oculto e Elias
8. Vocabulário canônico
9. Cuidados narrativos
