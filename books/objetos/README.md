# Objetos — Histórias que se Cruzam

Fichas individuais por objeto. Cada ficha controla forma, posição, estado, aparições e conexões — para que **um objeto descrito em um livro permaneça idêntico** em qualquer outro.

**Master tracker:** [`../biblia-continuidade-objetos.md`](../biblia-continuidade-objetos.md).

## Princípio essencial

> Detalhe descrito vira canônico. Posição fixa é fixa. Estado pode evoluir, mas a evolução é registrada.

## Estrutura de cada ficha

1. Identidade
2. Descrição visual canônica
3. Estado, posição, marcas
4. Aparições por livro
5. Quem toca / vê
6. Conexões com outros objetos
7. Conexões com Antigo, Personagem Oculto e Elias
8. Pistas plantadas
9. Cuidados narrativos
