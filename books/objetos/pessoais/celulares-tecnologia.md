# Celulares e tecnologia

> **Tier 4 — Pessoais.**
> Categoria que controla **toda tecnologia moderna** na série — celulares, computadores, redes sociais, aplicativos.

## 1. Identidade

A série se passa no presente — personagens usam tecnologia. Mas a obra **nunca nomeia marcas reais**. Toda tecnologia entra como **descrição genérica**, mantendo a obra universal e atemporal.

## 2. Regra de nomenclatura

| Em vez de | Usar |
|---|---|
| WhatsApp / iMessage | *“o aplicativo de mensagens”* / *“mensagens”* |
| Instagram / TikTok | *“a rede social”* / *“a rede”* |
| Spotify / YouTube | *“o aplicativo de música”* |
| Uber / 99 | *“o aplicativo de corrida”* |
| iPhone / Samsung | *“o telefone”* / *“o celular”* |
| Apple / Samsung / etc. | (nada — não nomear) |
| Notebook / MacBook | *“o computador portátil”* / *“o notebook”* |
| Gmail / Outlook | *“o e-mail”* |
| Google / busca | *“pesquisar”* / *“procurar”* |
| Stories | *“publicações”* / *“postagens curtas”* |
| Reels / vídeo curto | *“vídeo curto”* / *“vídeo da rede”* |

## 3. Personagens e tecnologia

| Personagem | Relação canônica |
|---|---|
| Lara | Telefone com tela rachada; redes sociais usadas para validação; pode bloquear contatos como gesto de transformação |
| Miguel | Telefone simples; uso funcional; pouco em redes |
| Rafael | Telefone caro (sem nomear), notebook fino, agenda eletrônica |
| Clara | Telefone funcional; muitos contatos; mensagens curtas |
| Helena | Telefone básico; aprendeu tarde; e-mail raramente |
| Theo | Telefone caro (sem nomear); muitas redes; ativo em vídeos curtos |
| Nina | **Telefone usado profissionalmente**; câmera; gravador; apps de transcrição (genéricos) |
| Sofia | Adolescente — múltiplas redes; mensagens em vários apps; pressão social digital |
| Elias | **Sem telefone moderno** ou com aparelho muito antigo (a definir) — sinal de pertencimento a outro tempo |
| Personagem Oculto | **Nunca tem tecnologia.** Sem telefone, sem fone, sem relógio digital |

## 4. Mensagens transcritas

Quando uma mensagem aparece transcrita no livro, segue formato fixo:

```
Mensagem recebida.
[texto da mensagem]
```

Sem emoji, sem nome de aplicativo, sem horário detalhado em alguns casos. **Texto direto.**

## 6. Conexões

- **[Bilhetes e anotações](../circulantes/bilhetes-e-anotacoes.md):** mensagens transcritas entram nesta categoria narrativamente.
- **[Foto antiga](../circulantes/foto-antiga.md):** Nina pode fotografar com câmera profissional ou com o celular.

## 7. Conexões com Antigo, Oculto, Elias

- **Antigo:** **contraste explícito.** O Antigo é manuscrito, antigo, físico. Tecnologia digital é o oposto. Quando os dois aparecem na mesma cena, o contraste é canônico.
- **Oculto:** **nunca usa tecnologia.** Esse silêncio é parte da regra.
- **Elias:** **mínima tecnologia.** Pode ter aparelho muito antigo ou nada — escolha grande sobre seu pertencimento.

## 9. Cuidados narrativos

- **Atemporalidade controlada.** A obra deve poder ser lida em 5–10 anos sem parecer datada — por isso, sem marcas, sem termos passageiros (*“dar match”*, *“viralizar”* etc., usados com parcimônia).
- **Tecnologia não pode dominar a cena.** Ela é instrumento, nunca tema.
- **Mensagens transcritas** seguem formato fixo (§4).
- **Sem ícones / emojis** no texto.
- **Vícios em redes** podem aparecer (Theo, Sofia), tratados com maturidade — mostrar a pressão e o vazio, sem caricaturar.

## 10. Decisões pendentes

- Elias usa aparelho antigo ou nenhum?
- Nome interno (não narrativo) do app de mensagens, para coerência interna do autor.
- Algum personagem-semente está vinculado a uma rede específica?
