# Árvore antiga

> **Tier 3 — Cenário.**
> A maior árvore do **parque**. Sob ela, o **banco do Oculto**.

## 1. Identidade

Árvore antiga, maior do parque. Tronco grosso, copa larga, raízes que erguem a calçada em alguns trechos. **Marca canônica:** **inscrição apagada no tronco** — possivelmente um nome, uma data ou um símbolo, riscado com canivete há décadas.

## 2. Descrição visual canônica

| Atributo | Estado canônico |
|---|---|
| Espécie | A definir — sugestão: árvore frondosa de copa larga e tronco escuro (sem nomear espécie regional reconhecível) |
| Altura | A maior do parque |
| Tronco | Grosso, com cicatriz de uma inscrição apagada |
| Copa | Ampla — protege o banco |
| Raízes | Erguem a calçada em alguns trechos |
| Folhagem | Verde escuro; em outono, perde algumas folhas |
| Sons | Vento entre as folhas como sinal canônico |

## 3. A inscrição apagada

| Atributo | Canon |
|---|---|
| Forma | Letras / números riscados a canivete |
| Estado | Quase ilegíveis |
| Conteúdo | A definir — possivelmente um nome (*Elias*?), uma data, ou um símbolo ligado ao Antigo |
| Profundidade | Suficiente para sobreviver, mas com bordas alisadas pelo tempo |

> **Decisão grande:** se a inscrição é o nome de Elias, ela é a **prova material** do peso simbólico do parque para ele. Esta confirmação cabe em L9.

## 8. Cuidados narrativos

- **Sem antropomorfização.** A árvore não “sente”. É testemunha visual.
- **Inscrição é canônica.** Quando aparece, mesma posição, mesmo estado.
- **Sem nomear espécie regional reconhecível** (jacarandá, ipê, mangueira). Manter genérico — *“a árvore antiga”*, *“a árvore maior”*, *“a árvore de copa larga”*.
- **A inscrição é descoberta em camadas.** L1 menciona; L7 amplia; L9 lê. Cada livro acrescenta um detalhe — **sem contradizer** os anteriores.
