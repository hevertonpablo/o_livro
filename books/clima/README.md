## Estrutura de cada ficha

1. Identidade
2. Características climáticas / sensoriais
3. Função simbólica
4. Cheiros, luzes, sons canônicos
5. Conexão com personagens
6. Conexão com cenários
7. Conexão com Antigo, Oculto, Elias
8. Em quais livros aparece
9. Cuidados narrativos

## Princípios gerais

- Clima nunca é decoração. Sempre tem função.
- Cheiros do Oculto: **café, chuva, madeira** — pelo menos um por aparição.
- Cenas compartilhadas entre livros têm clima idêntico.
- Sinais climáticos recorrentes (trovão sem chuva, rajada que vira página, sino distante) só em momentos-chave.
